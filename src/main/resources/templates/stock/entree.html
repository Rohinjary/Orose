Règles de gestion

Les entrées de stock sont gérées selon la méthode FIFO (First In, First Out).

Une entrée d'aliments génère automatiquement une dépense dans le module Finances (catégorie ALIMENTATION).

Bon à savoir

Les entrées de médicaments sont gérées via ce formulaire. Les entrées d'aliments s'effectuent depuis le module Nourrissage.
